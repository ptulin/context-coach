# Changelog

## v1.0.0 — March 2026

Initial release.

### Features
- Project Kickoff Protocol (3 scoping questions + SESSION BRIEF)
- Red Flag detection: scope creep, re-explanations, giant unscoped files, multi-topic pile-ons
- Yellow Flag detection: tangents, verbose requests, repeated tool calls
- Response style defaults (answer-first, concise, no preamble)
- Checkpoint protocol every ~4 major steps with context health indicator
- Fresh Conversation Handoff summary template
- Session Efficiency Report with side-by-side coached vs. uncoached comparison
- API call awareness and batching nudges

### Known limitations
- Token estimates are behavioral estimates, not real measurements
- Skill cannot read actual Claude token counts (API limitation)
- Best results when activated at session start, not mid-session

### Roadmap for v2
- Pre-session project brief templates
- Obsidian MCP integration for persistent project memory
- Model routing hints
- Skills/extensions overhead awareness
